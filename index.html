En résumé

  • MySQL est un SGBDR Système de gestion de base de données.
  • Le langage SQL est utilisé pour la communication entre le client et le serveur.
  • Dans une BDD les données sont représentées sous forme de table.
  • Dans une table nous avons des colonnes,chaque donnés sera introduite sous forme de ligne dans la table.

SGBDR/SGBD

  • Un système qui permet de gerer une BDD /// CRUD Create ,Read ,Update ,Delete.
  • Le logiciel client permet de dialoguer avec le serveur avec le langage SQL.
  • MySQLStructured Query Language.
  • Date de naissance 1994
  • Open source.
  • Concurrents:Oracle, MS Access SQLite PostgreSQL.

Requête de base

  • Créer une bdd CREATE DATABASE (basetest)= nom de la table character SET 'utf8';
  • Supprimer une BDD DROP DATABASE IF EXISTS basetest
  • Créer une table: CREATE TABLE t_utilisateurs INSERT INTO t_utilisateurs (id, nom, prenom, email, date_naissance, pays, ville, code_postal, nombre_achat) VALUES (NULL, 'Hugo', 'Victor', 'victor.hugo@france.fr', '1905-02-02', 'France', 'Paris', '75019', NULL);
  • Insérer dans une table: INSERT INTO t_utilisateurs
  • Afficher le contenu dans une table SELECT * FROM t_utilisateurs
  • Afficher et trier le contenu de la table SELECT * FROM t_utilisateurs ORDER BY nom ASC
  • Effacer un enregistrement: DELETE FROM t_utilisateurs WHERE nom='chirac'

Types de données pour créer un tableau

  • CREATE TABLE t_utilisateurs ( id INT PRIMARY KEY NOT NULL AUTO_INCREMENT, nom VARCHAR(100), prenom VARCHAR(100), email VARCHAR(255), date_naissance DATE, pays VARCHAR(255), ville VARCHAR(255), code_postal VARCHAR(5), nombre_achat INT )
